# Product Roadmap

## Phase 1: MVP

- **Web UI** — Form-based interface for entering and reviewing weekly timesheet data
- **Parse template** — Read the NAS CSV template and understand its fixed layout
- **Fill from input** — Accept shift/schedule data and populate the timesheet
- **Overtime detection** — Automatically calculate and flag overtime hours worked
- **Dual output** — Generate both `.csv` and `.xlsx` files matching the NAS format
- **Email to employee** — Send completed timesheet to the employee for review before submission

## Phase 2: Post-Launch

- **Roster import** — Import roster/schedule data from external sources (e.g., PDF, system export)
- **Batch generation** — Generate timesheets for multiple weeks or multiple staff at once
- **Payroll submission** — Forward approved timesheets directly to ambulance.payroll@hse.ie
- **User accounts** — Save staff details and preferences across sessions
