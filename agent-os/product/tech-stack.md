# Tech Stack

## Runtime

- **Node.js** with **TypeScript**

## Framework

- **Next.js** — Full-stack React framework (App Router)
- **React** — UI components

## Libraries

- **csv-parse / csv-stringify** — CSV reading and writing
- **ExcelJS** — Excel (.xlsx) generation with formatting support
- **nodemailer** — Email sending for timesheet submission
- **zod** — Input validation and type-safe parsing

## Build & Dev

- **Next.js built-in** — Dev server, bundling, and production builds

## Testing

- **vitest** — Fast TypeScript-native test runner
