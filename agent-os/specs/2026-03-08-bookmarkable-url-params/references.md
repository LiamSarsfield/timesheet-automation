# References: Bookmarkable URL Parameters

## APIs Used

- [`URLSearchParams`](https://developer.mozilla.org/en-US/docs/Web/API/URLSearchParams) — parse and build query strings
- [`history.replaceState`](https://developer.mozilla.org/en-US/docs/Web/API/History/replaceState) — update URL without navigation
- [`window.location.search`](https://developer.mozilla.org/en-US/docs/Web/API/Location/search) — read current query string

## Pattern References

- Debounced URL sync is a common pattern in filter/search UIs
- Similar to Next.js shallow routing but without the framework overhead
